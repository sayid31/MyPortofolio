Selamat datang di repositori *source code* portofolio personal saya. Web ini dirancang sebagai Single Page Application (SPA) yang interaktif untuk menampilkan rekam jejak, proyek *engineering* (Web & IoT), dan spesialisasi teknis saya.

🔴 **[LIHAT LIVE DEMO DI SINI](https://sayid31.github.io/MyPortofolio/)** 🔴

## ✨ Fitur Utama (UI/UX)

*   **vCard Architecture:** Navigasi intuitif berbasis *tab* untuk memisahkan profil utama, latar belakang, dan etalase proyek.
*   **Terminal/Hacker Aesthetic:** Menggunakan elemen *Deep Dark Mode* (`zinc-950`), aksen neon (Cyan/Emerald), dan tipografi *monospace* yang merepresentasikan lingkungan kerja seorang *developer*.
*   **Fully Responsive:** Tampilan dioptimalkan secara penuh untuk resolusi *mobile*, *tablet*, maupun *desktop*.
*   **Interactive Project Showcase:** Menampilkan detail arsitektur dari proyek *Fullstack* (Node.js/React) dan perangkat keras IoT.

## 🛠️ Tech Stack & Ekosistem

*   **Framework:** React.js (Fungsional & Hooks)
*   **Build Tool:** Vite (Untuk *rendering* dan HMR yang sangat cepat)
*   **Styling:** Tailwind CSS (Utilitas CSS modern)
*   **Icons:** Lucide React
*   **Deployment:** GitHub Pages (Otomatisasi CI/CD via `gh-pages`)
